# Vynce

A premium Material 3 music player for Android, powered by JioSaavn.

## Features
- 🎵 Stream 100M+ songs at 320kbps via JioSaavn
- 📱 Beautiful Material 3 dark UI with dynamic colors  
- 🎨 Rotating vinyl album art + audio visualizer
- 📂 Local music playback (MP3, FLAC, OGG, AAC)
- 📊 Listening stats dashboard
- 💿 Synchronized lyrics (LRC, TTML)
- 🌙 Sleep timer, audio normalization
- 📻 Last.fm scrobbling
- 🚗 Android Auto support
- ✈️ Works in India perfectly — no auth required
- 📥 Download support
